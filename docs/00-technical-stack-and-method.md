# Bruno Simon Portfolio 逆向拆解：技术栈与方法

## 目标

对 `https://bruno-simon.com/` 做黑盒与公开资源层面的工程逆向，输出可验证的技术栈、渲染架构、交互系统、资产管线、性能策略与复刻路线。

## 调研技术栈

- 页面与交互：真实浏览器、DOM / Canvas 状态、键鼠与触控路径。
- 网络与资产：HTTP 头、HTML / JS / CSS 入口、资源清单、模型/纹理/音频格式。
- 实现线索：产物指纹、公开源码/作者资料与运行时证据交叉验证。
- 性能：首屏加载、资产分批、渲染循环、物理步进、移动端退化。

## 拟定复刻技术栈（待证据校正）

- TypeScript + Vite
- Three.js / WebGL
- Rapier 或 cannon-es 物理引擎
- Blender → glTF/GLB + KTX2/Draco 资产管线
- Howler.js 或 Web Audio API
- HTML/CSS 覆盖层用于加载、帮助、导航与无障碍补充

## 证据标记

最终报告将区分：

- 已验证：可由运行时、网络或作者公开资料直接支持。
- 高概率推断：由产物特征和运行行为推导，但未获得源码级证据。
- 复刻建议：针对当前工程的实施选择，不声称与原站完全相同。
